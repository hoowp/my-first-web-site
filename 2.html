Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications. Web browsers receive HTML documents from a web server or from local storage and render them into multimedia web pages . HTML describes the structure of a web page semantically and originally included cues for the appearance of the document. 1. color 2. direction 3. letter-spacing 4. word-spacing 5. text-indent 6. text-align 7. text-decoration 8. text-transform 9. line-height 10. text-shadow
